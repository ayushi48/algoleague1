## 📌 Overview

**AlgoLeague** is a full-stack online coding platform designed for programmers to practice coding problems, run code instantly, and submit solutions with real-time evaluation.

The platform supports **multiple programming languages** and provides a seamless experience for both users and administrators through secure authentication, role-based access control, and an interactive coding environment.

## 🛠️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/ayushi48/AlgoLeague.git
cd AlgoLeague
```

### 2️⃣ Install Dependencies

**Frontend**
```bash
cd frontend
npm install
```

**Backend**
```bash
cd backend
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file inside the `backend` directory:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
JWT_REFRESH_SECRET=your_refresh_secret
REDIS_URL=your_redis_url
JUDGE0_API_KEY=your_judge0_api_key
```

### 4️⃣ Run the Project

**Backend**
```bash
npm run dev
```

**Frontend**
```bash
npm run dev
```
