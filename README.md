# 📌 Overview

**AlgoLeague** is a full-stack online judge platform designed for programmers to practice coding, execute solutions in real time, and receive detailed evaluation reports.


The platform offers a competitive-programming-inspired experience with secure authentication, role-based access control, real-time code execution, analytics dashboards, and comprehensive submission tracking.


## ⚡ Tech Stack

<div align="center">

| Frontend     | Backend    | Database    | Security          | Services           |
| ------------ | ---------- | ----------- | ----------------- | ------------------ |
| React        | Node.js    | MongoDB     | JWT               | Judge0 API         |
| Tailwind CSS | Express.js | Mongoose    | RBAC              | Redis              |
| React Router | REST APIs  | Redis Cache | HTTP-only Cookies | Session Management |

# 🛠️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/ayushi48/AlgoLeague.git
cd AlgoLeague
```

## 2. Install Dependencies

```bash
# Frontend
cd frontend
npm install

# Backend
cd backend
npm install
```

---

## 3. Configure Environment Variables

Create `.env` inside backend directory.

```env
MONGO_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

JWT_REFRESH_SECRET=your_refresh_secret

REDIS_URL=your_redis_url

JUDGE0_API_KEY=your_judge0_api_key
```

---

## 4. Run Development Server

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```
